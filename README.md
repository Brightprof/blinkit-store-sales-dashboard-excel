# Blinkit Sales Analysis Dashboard (Excel)

## Project Overview
This project analyzes Blinkit store sales data to uncover insights on
sales performance, outlet size, location, item categories, and customer preferences.
The analysis was performed using Microsoft Excel with a focus on data cleaning,
analysis, and dashboard storytelling.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Power Query
- Excel Formulas (SUMIFS, COUNTIFS, IF)
- Data Visualization & Dashboard Design

## Key Metrics
- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8,523
- Average Rating: 4.0

## Analysis Highlights
- Outlet size significantly impacts total sales, with large outlets contributing the highest revenue.
- Tier 1 locations generated the highest sales compared to Tier 2 and Tier 3.
- Fruits and Vegetables and Snack Foods are the top-performing item categories.
- Regular fat content items account for a higher percentage of total sales.
- Sales trends show steady growth with a peak around 2018.

## Project Workflow
1. Data cleaning and preprocessing
2. Handling missing values and inconsistencies
3. Creating calculated columns and KPIs
4. Building pivot tables for analysis
5. Designing an interactive Excel dashboard with filters
